# shop-shop

## Purpose:
Refactor existing React application to use Redux to handle global state. Provide a functional ecommerce web app where users can browse products, manage carts, login or signup, and check out.

## Technology Used:
* GraphQL
* Apollo
* MongoDB
* React
* Redux
